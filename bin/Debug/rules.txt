Вам необходимо угадать трехзначное число, загаданное компьютером.

Количество быков означает, сколько цифр из введенного Вами числа
совпадают и по значению, и по позиции.
Количество коров показывает сколько цифр совпадают с загаданным
числом только по значению.

Например, компьютер загадал число 357.
-Вы вводите 751. Количество быков равно 1, количество коров - 1.
- 333. Количество быков равно 1, количество коров - 2.
- 397. Количество быков равно двум, количество коров 0.
- 573. Быки - 0. Коровы - 3. 

Чтобы подробнее разобраться, начните играть.
